🧪 Chem Quiz Cards

10 карточек: формальное название + картинка, переворот = тривиальное название.

Ключи API

Сохраняются только в этом браузере (localStorage). Для GitHub Pages можно подставить значения при сборке: замени в файле строки __CHEM_BUILD_MISTRAL__ и __CHEM_BUILD_PIXAZO__ на ключи (например, шагом workflow с sed); пользовательский ввод ниже тогда необязателен.

Набор карточек

Свой список: по строке или через запятую. Русский режим: формальное | тривиальное. Режим ИЮПАК (галочка выше): английские названия; после | — бытовое имя на английском (необязательно). Неполные пары добиваются переводом и при необходимости Mistral + CID из PubChem.

Изображения

Тип `structure2d` доступен не для всех веществ. В этом случае карточка покажет название без картинки.
